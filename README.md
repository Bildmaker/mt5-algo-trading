## Lokaler Workflow

1. Aenderungen lokal im Projektordner machen.
2. Mit `git status` pruefen, was geaendert wurde.
3. Mit `git add ...` die gewuenschten Dateien vormerken.
4. Mit `git commit -m "..."` einen nachvollziehbaren Stand speichern.
5. Mit `git push` nach GitHub hochladen.

Details stehen in `bildmaker/metatrader5/TT_ORB_2026_15M_volume/docs/github-workflow.md`.
